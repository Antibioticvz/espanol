# Руководство пользователя: Combine + Audio Learner

**Версия:** 1.0  
**Дата:** 21 июля 2026

Это практическое руководство для преподавателя (вы будете и создавать уроки в Combine, и учить фразы в Audio Learner на iPhone).

---

## Быстрый старт

### Что вам потребуется
- **macOS** с установленным Combine (скачайте `.dmg`)
- **iPhone** с установленным Audio Learner
- **Текстовый файл** с уроком (в простом формате разметки)
- **ElevenLabs API-ключ** (опционально; без него Combine работает бесплатно в режиме `mock_say`)

---

## Часть 1: Combine на Desktop

### 1.1 Первый запуск Combine

1. **Скачайте и установите** Combine-1.0.0.dmg
   - Откройте DMG
   - Перетащите Combine.app в папку Applications
   - Запустите Combine

2. **При первом открытии** Combine попросит:
   - ✓ Выбрать папку для сохранения уроков (по умолчанию `~/lessons`)
   - ✓ Ввести ElevenLabs API-ключ (**или оставить пусто**, если будете использовать mock-режим)

### 1.2 Два режима работы

#### Режим 1: Mock (бесплатный)

Combine генерирует уроки через встроенный голос macOS (`say`). Достаточно для проверки структуры и UX, но качество зависит от голоса системы.

**Как включить:**
- Оставьте поле API-ключа **пустым**
- Combine автоматически переключится в режим `mock_say`
- На экране настроек будет отмечено «🆓 Mock-режим (бесплатно)»

**Преимущества:**
- Бесплатно, неограниченно
- Полнофункциональна работа Combine
- Хорошо для разработки и тестирования

#### Режим 2: ElevenLabs (платный, лучшее качество)

Combine использует реальный API ElevenLabs для синтеза профессионального качества.

**Как получить ключ:**
1. Перейдите на https://elevenlabs.io
2. Создайте аккаунт (Free или Pay-as-you-go)
3. Перейдите в Settings → API Keys
4. Скопируйте ваш ключ (начинается с `sk-`)

**Как включить:**
- На экране импорта → [Параметры]
- Поле «API Key» → вставьте ваш ключ
- Combine проверит подключение (кнопка «Тест») и сохранит в Keychain

**Стоимость:**
- ~$0.10 за 1000 символов (Multilingual v2)
- ~$0.05 за 1000 символов (Flash v2.5)
- Весь курс (50 тем) ≈ $37–75

### 1.3 Тестовая генерация (очень важно!)

**Зачем:** проверить, что ключ работает, выбрать подходящий голос, оценить стоимость — всё на короткой фразе (~25 символов, ~$0.003).

**Как сделать:**

1. На экране настроек найдите секцию **«Тестовая генерация»**
2. Поле по умолчанию: `¿Cómo te llamas?` → `Как тебя зовут?`
3. Выберите голос (Pablo для испанского, Masha для русского)
4. Нажмите **«🎙 Сгенерировать тест»**
5. Через 2–5 сек услышите результат
6. Если качество нравится → можно генерировать весь урок

**Результат:**
- ✓ Ключ работает (если не было ошибки)
- ✓ Голоса выбраны правильно
- ✓ Видите реальную стоимость (вверху экрана: «Тест: $0.003»)

## Часть 2: Подготовка урока

### 2.1 Формат текста

Создайте текстовый файл (`.txt` или `.md`) вот в таком формате:

```
#TOPIC 4 | Рассказ о себе

##BLOCK verb_group | Кто я — происхождение и факты

#WORD llamarse | зваться
Me llamo Victor. | Меня зовут Виктор.
¿Cómo te llamas tú? | Как тебя зовут?

##BLOCK phrase_group | Ходовые фразы

#CATEGORY Первое знакомство
Mucho gusto, soy Victor. | Очень приятно, я Виктор.

##BLOCK vocabulary | Ключевая лексика
el programador | программист
la familia | семья

##BLOCK story | Короткий рассказ
ES: Me llamo Victor...
RU: Меня зовут Виктор...
```

**Основные правила:**
- ✓ Разделитель между текстом и переводом: ` | ` (пробел-тире-пробел)
- ✓ Только один разделитель на строку
- ✓ Пустые строки между блоками (они автоматически пропускаются)
- ✓ Чувствителен к формату #TOPIC и ##BLOCK

**Подробнее:** см. документ `LESSON_FORMAT.md`

### 2.2 Загрузка в Combine

1. **Экран 1: Импорт текста**
   - Нажмите **[📄 Загрузить файл]** (выберите `.txt`)
   - Или **[Вставить вручную]** (скопируйте текст в поле)
   - Combine парсит в реальном времени, показывает:
     - ✓ «Тема найдена: "Рассказ о себе"»
     - ✓ «Блоков: 4, Фраз: 81, Ошибок: 0»

2. **Проверьте валидацию:**
   - ✗ Красным: ошибки парсинга (номер строки + описание)
   - ⚠️ Жёлтым: предупреждения (например, пустая категория)
   - ✓ Зелёным: готово к генерации

3. Нажмите **[Далее →]** если всё окей

## Часть 3: Генерация урока

### 3.1 Экран настроек

**Левая панель — ElevenLabs параметры:**

| Параметр | Значение | Совет |
|----------|---------|-------|
| **API Key** | sk-••••• | Видно только для проверки (скрыто) |
| **Модель** | Multilingual v2 или Flash v2.5 | Flash дешевле в 2 раза, но медленнее |
| **Голос ES** | Pablo, Diego, Maria, Sofia | Выберите, прослушав тест (кнопка 🔊) |
| **Голос RU** | Masha, Sasha, Natasha, Aleksandr | Фиксируйте выбор: его потом нельзя менять |
| **Stability** | 0.0–1.0 (по умолчанию 0.5) | Высокая = монотонная, низкая = вариативная |
| **Similarity Boost** | 0.0–1.0 (по умолчанию 0.75) | Насколько точно копировать голос |

**Правая панель — параметры генерации:**

| Параметр | Значение | Совет |
|----------|---------|-------|
| **Папка вывода** | ~/lessons | Куда сохранять готовые уроки |
| **Параллельные запросы** | 1–5 (по умолчанию 3) | Больше = быстрее, но риск ошибок (429) |
| **Макс попыток** | 1–10 (по умолчанию 3) | Сколько раз повторять при ошибках |
| **Delay** | 100 ms | Пауза между запросами (выше = медленнее, но безопаснее) |
| **Timeout** | 30 сек | Максимум ждать ответа API |

**Калькулятор стоимости** (внизу):
```
Модель: Multilingual v2 ($0.10 / 1000 сим)
Языков: 2
Фраз всего: 81

Символов:
  ES: 3 737
  RU: 3 799
  ИТОГО: 7 536

Цена: 7 536 × $0.10 / 1000 = $0.75
```

⚠️ Это оценка (±5%). Реальная цена может отличаться.

### 3.2 Запуск генерации

1. **Убедитесь:** API-ключ работает (кнопка «Тест» должна показать ✓)
2. **Нажмите [Генерировать →]**

### 3.3 Экран генерации

Вы увидите прогресс в реальном времени:

```
Генерация: Рассказ о себе

Статус: [████████════] 66% (54 / 81 фраз)

Текущий элемент:
💬 04-b1-llamarse-02 (ES)
   "¿Cómo te llamas tú?"
   [████████───] 3.2 / 4.8 сек

Статистика:
Скорость: 12.4 фраз/мин
Осталось: ~3 мин
Успешно: 54 | Ошибок: 0 | Пропущено: 0
Потрачено: $0.41 / $0.75 (55%)
```

**Дерево блоков (разворачиваемое):**
- ▼ BLOCK 1: verb_group (20 фраз) [████████──]
  - ▼ llamarse (5 фраз) [██████████]
    - ✓ 04-b1-llamarse-01 ES (1.2 сек)
    - ✓ 04-b1-llamarse-01 RU (1.1 сек)
    - ⏱ 04-b1-llamarse-02 ES [generating...]
  - ▼ tener (6 фраз) [██████░░░░]

**Кнопки управления:**
- ⏸ **Пауза** — остановить, возобновить позже
- ⏹ **Отмена** — остановить без сохранения (потерять незаконченное)

### 3.4 При ошибке 429 (лимит)

Combine автоматически:
1. Обнаруживает 429 Too Many Requests
2. Ждёт (exponential backoff: 1s → 2s → 4s)
3. Повторяет (макс 3 попытки)
4. Если не помогло → помечает фразу как `failed`

**Что делать:**
- ✓ Подождите (лимит восстановится за ~1 час)
- ✓ Нажмите [Возобновить] → Combine продолжит с `failed` фраз

### 3.5 Завершение

Когда все фразы сгенерированы:

```
✓ Урок "Рассказ о себе" готов!

Папка: ~/lessons/04-hablar-de-mi-mismo/
Статистика:
  • 81 фраза + 15 слов + 1 рассказ
  • 162 аудиофайла
  • 34.2 МБ
  • Затрачено: $0.74

[🎵 Проиграть] [📂 Открыть в Finder] [Далее →]
```

Нажмите **[📂 Открыть в Finder]** — папка `04-hablar-de-mi-mismo/` будет выделена.

## Часть 4: Экспорт на iPhone

### 4.1 Экран библиотеки

1. На главном окне Combine → вкладка **Библиотека**
2. Найдите нужный урок (например, «Рассказ о себе»)
3. Нажмите **[⋯ Ещё]** → **«Экспорт ZIP → Downloads»**

**Результат:**
- Скачивается файл `lesson-04-hablar-de-mi-mismo.zip`
- Размер: ~34 МБ
- Структура внутри:
  ```
  lesson.json (манифест)
  audio/
  ├── es/ (81 MP3)
  └── ru/ (81 MP3)
  ```

### 4.2 Три способа передачи на iPhone

#### Способ 1: AirDrop (самый быстрый)

1. На macOS: Finder → Downloads → `lesson-04.zip`
2. Right-click → AirDrop
3. Выберите свой iPhone
4. На iPhone: примет файл и откроет в Audio Learner

#### Способ 2: iCloud Drive

1. На macOS: Finder → iCloud Drive → создайте папку `AudioLearner`
2. Перетащите `lesson-04.zip` в эту папку
3. На iPhone: Files.app → iCloud Drive → AudioLearner → lesson-04.zip
4. Нажмите на файл → откроется в Audio Learner

#### Способ 3: Email / облако

1. На macOS: отправьте себе `lesson-04.zip` по email
2. На iPhone: откройте email → tap на вложение → "Copy to Audio Learner"

## Часть 6: Обучение на iOS

### 6.1 Создание сессии

1. **Уроки** → нажмите на урок → **[▶ Играть]**
2. **Выбор фраз:**
   - Фильтр по статусу: Все / Выучено / В процессе / Учу
   - Выберите нужные фразы (checkboxes)
   - Нажмите **[Далее →]**

3. **Конфигурация сессии:**
   - **Повторения:** 5 (сколько раз повторить каждую фразу)
   - **Скорость:** 0.5x (замедленная)
   - **Пауза:** 7 сек (между повторениями)
   - **Режим:** Один раз (все и готово)
   - **Lock screen:** Оригинал + Перевод

4. **Расчёт:** «Ожидаемое время: 8 мин 45 сек»

5. **[▶ Начать сессию]**

### 6.2 Плеер

```

🇪🇸 "¿Cómo te llamas tú?"
🇷🇺 "Как тебя зовут?"

[🔊 Воспроизведение: 2.1 / 3.2 сек]
[████████───────────────] 2.1 / 3.2 sec

[⏮ Назад] [⏸ Пауза] [▶ Плей] [⏭ Далее]

[❤️ Избранное] [🔁 Повторить фразу]
```

**Кнопки:**
- **▶ / ⏸** — плей / пауза

## Часть 7: Поиск и устранение неполадок

### Combine (Desktop)

| Проблема | Решение |
|----------|---------|
| **«Invalid API Key»** | Проверьте ключ на https://elevenlabs.io/account. Удалите из Keychain (Cmd+,), введите заново. |
| **«429 Too Many Requests»** | Лимит восстановится за ~1 час. Нажмите [Возобновить] — Combine продолжит. |
| **«No space left on device»** | Проверьте свободное место (`df -h`), очистите другие файлы или выберите другую папку. |
| **Генерация зависла на 50%** | Нажмите [Отмена], потом откройте урок заново → [Возобновить]. |
| **Парсер ругается на формат** | Проверьте разделитель: должен быть ровно ` | ` (пробел-тире-пробел). |
| **ZIP не создаётся** | Папка ~/lessons может быть недоступна. Проверьте права: `chmod u+w ~/lessons`. |
| **Цена слишком высока** | Используйте Flash вместо Multilingual (дешевле в 2 раза). Или сократите фразы. |

### iOS (Audio Learner)

| Проблема | Решение |
|----------|---------|
| **«ZIP не импортируется»** | Проверьте что это правильный ZIP от Combine. Попробуйте распаковать вручную и импортировать заново. |
| **«Аудио не воспроизводится»** | Проверьте что Documents/AudioLearner/lessons/*/audio/ содержит MP3 файлы. Перезагрузите приложение. |
| **«App падает при импорте»** | Закройте другие приложения, перезагрузите iPhone. Если сохраняется — переустановите app. |
| **«Lock screen не обновляется»** | Перезагрузите сессию. В Параметры → Аудио → проверьте что вибрация и звук включены. |
| **«Прогресс потерян»** | Восстановите из резервной копии: Параметры → Данные → [Восстановить из backup]. |
| **«CoreData ошибка»** | Параметры → Данные → [Очистить все]. Переустановите урок. Прогресс потеряется (извините). |

### Общие вопросы

**Q: Можно ли менять голоса после генерации?**
A: Нет, выбор голоса фиксируется в lesson.json. Если хотите другой голос — переделайте генерацию (это добавит затраты).

**Q: Как обновить урок без потери прогресса?**
A: Используйте [Обновить] при импорте (а не [Заменить]). Это скопирует новые аудиофайлы, но прогресс останется.

**Q: Сколько времени занимает генерация 81 фразы?**
A: 15–20 минут (зависит от лимитов ElevenLabs). В фоне можно свернуть Combine — процесс продолжится.

**Q: Может ли быть ошибка парсера из-за переводов?**
A: Нет, парсер не смотрит на содержание. Проверяет только формат: наличие разделителя ` | ` и структуру блоков.

**Q: Что если оставить API-ключ пустым?**
A: Combine всегда работает в mock-режиме (бесплатно). Если позже захотите платный — введите ключ в параметры.

### Обучение SRS (Spaced Repetition)

Каждая фраза проходит три состояния:

1. **learning** → 🆕 новая
   - Условие: никогда не повторяли или мало повторяли
   - Переход: после 3 повторений → inProgress

2. **inProgress** → 🔄 в процессе
   - Условие: повторяли, но не достаточно
   - Переход: после 8 повторений → mastered

3. **mastered** → ✅ выучено
   - Условие: хорошо выучена
