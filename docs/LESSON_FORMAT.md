# Справочник: Формат входного файла урока

**Версия:** 1.0  
**Дата:** 21 июля 2026

Полное описание синтаксиса и правил разметки для подготовки уроков в Combine. Используйте этот документ как справочник при написании новых тем.

---

## 1. Обзор структуры

### 1.1 Иерархия элементов

```
#TOPIC (ровно 1)
  └─ ##BLOCK (1 или более)
      ├─ #WORD или #CATEGORY (опционально)
      │  └─ фраза ES | RU (1 или более)
      │
      └─ #WORD или #CATEGORY (опционально)
         └─ фраза ES | RU (1 или более)
```

### 1.2 Четыре типа блоков

| Тип | Назначение | Пример |
|-----|-----------|---------|
| **verb_group** | Глаголы, сгруппированные по инфинитиву | "llamarse", "tener" |
| **phrase_group** | Устойчивые фразы, сгруппированные по категориям | "Первое знакомство", "О работе" |
| **vocabulary** | Плоский список слов (без группировки) | "el programador", "la familia" |
| **story** | Один сплошной текст (рассказ или диалог) | Длинный монолог/диалог |

---

## 2. Синтаксис каждого элемента

### 2.1 TOPIC (тема урока)

**Формат:**
```
#TOPIC <номер> | <название на русском>
```

**Правила:**
- ✓ Ровно одна тема на файл
- ✓ Номер: 1–50, целое число
- ✓ Разделитель: ` | ` (пробел-тире-пробел)
- ✓ Название: на русском, любая длина
- ✗ Не может быть пустым

**Примеры:**
```
#TOPIC 1 | Приветствие и знакомство
#TOPIC 4 | Рассказ о себе
#TOPIC 50 | Будущее и планы
```

**Ошибки:**
```
❌ TOPIC 4 | Рассказ о себе        (нет #)
❌ #TOPIC 4 Рассказ о себе         (нет |)
❌ #TOPIC Четыре | Рассказ о себе  (номер не число)
❌ #TOPIC 4 |                      (название пусто)
```

---

### 2.2 BLOCK (блок внутри темы)

**Формат:**
```
##BLOCK <тип> | <название на русском>
```

**Параметры:**

| Параметр | Значения | Примеры |
|----------|---------|---------|
| **тип** | verb_group, phrase_group, vocabulary, story | `##BLOCK verb_group` |
| **название** | Любой текст на русском | `| Кто я — происхождение и факты` |

**Правила:**
- ✓ Один или больше блоков на тему
- ✓ Разделитель: ` | ` (пробел-тире-пробел)
- ✓ Название рекомендуется (помогает при изучении)
- ✓ Порядок блоков сохраняется (verb_group → phrase_group → vocabulary → story)

**Примеры:**
```
##BLOCK verb_group | Кто я — происхождение и факты
##BLOCK phrase_group | Ходовые фразы
##BLOCK vocabulary | Ключевая лексика
##BLOCK story | Короткий рассказ
```

**Ошибки:**
```
❌ ##BLOCK verb_group Кто я        (нет |)
❌ #BLOCK verb_group | Кто я       (одна #)
❌ ##BLOCK unknown_type | Блок     (неверный тип)
```

---

### 2.3 WORD (метка глагола в verb_group)

**Формат:**
```
#WORD <инфинитив> | <перевод на русском>
```

**Правила:**
- ✓ Только внутри блока `verb_group`
- ✓ Группирует фразы по инфинитиву
- ✓ Разделитель: ` | ` (пробел-тире-пробел)
- ✓ Инфинитив: испанский глагол в инфинитиве (llamarse, tener, trabajar)
- ✓ Перевод: русский перевод инфинитива

**Примеры:**
```
#WORD llamarse | зваться
#WORD tener | иметь (возраст)
#WORD trabajar | работать
```

**Ошибки:**
```
❌ #WORD llamarse зваться          (нет |)
❌ WORD llamarse | зваться         (нет #)
❌ #WORD | зваться                 (инфинитив пуст)
```

**После #WORD идут фразы:**
```
#WORD llamarse | зваться
Me llamo Victor. | Меня зовут Виктор.
¿Cómo te llamas tú? | Как тебя зовут?
```

---

### 2.4 CATEGORY (метка категории в phrase_group)

**Формат:**
```
#CATEGORY <название категории>
```

**Правила:**
- ✓ Только внутри блока `phrase_group`
- ✓ Группирует фразы по темам
- ✓ Название: любой текст на русском
- ✗ Нет разделителя `|` (в отличие от #WORD)
- ✓ После идут фразы

**Примеры:**
```
#CATEGORY Первое знакомство
#CATEGORY О работе
#CATEGORY Семья
```

**Ошибки:**
```
❌ #CATEGORY Первое знакомство |   (лишний |)
❌ CATEGORY Первое знакомство      (нет #)
❌ #CATEGORY                       (название пусто)
```

**После #CATEGORY идут фразы:**
```
#CATEGORY Первое знакомство
Mucho gusto, soy Victor. | Очень приятно, я Виктор.
Encantado de conocerte. | Приятно познакомиться.
```

---

### 2.5 Фраза (строка с текстом)

**Формат:**
```
<текст на испанском> | <текст на русском>
```

**Правила:**
- ✓ Ровно один разделитель ` | ` на строку
- ✓ Оба текста обязательны
- ✓ Можно использовать пунктуацию, апострофы, диакритические знаки
- ✓ Пробелы вокруг `|` важны (` | `, не `|`)
- ✓ Длина: от 1 символа до нескольких сотен

**Примеры:**
```
Me llamo Victor. | Меня зовут Виктор.
¿Cómo te llamas tú? | Как тебя зовут?
Todos me llaman Vic, para abreviar. | Все зовут меня Вик, для краткости.
el programador | программист
```

**С диакритикой (испанский):**
```
¿Cuántos años tienes tú? | Сколько лет тебе?
España | Испания
Trabajar a distancia. | Работать удалённо.
```

**Ошибки:**
```
❌ Me llamo VictorМеня зовут Виктор    (нет |)
❌ Me llamo Victor || Меня зовут       (два |)
❌ Me llamo Victor| Меня зовут         (нет пробела перед |)
❌ Me llamo Victor |Меня зовут         (нет пробела после |)
❌ Me llamo Victor |                   (русский текст пуст)
❌ | Меня зовут Виктор                (испанский текст пуст)
```

---

### 2.6 Story (рассказ)

**Формат:**
```
##BLOCK story | <название>

ES: <длинный текст на испанском...>
RU: <длинный текст на русском...>
```

**Правила:**
- ✓ Один или больше блоков story
- ✓ Всегда идут две строки подряд: ES: , потом RU:
- ✓ Текст может быть многострочный (продолжается до следующего ##BLOCK или конца файла)
- ✓ Тексты могут быть длинными (50–500+ символов)
- ✗ Нет разделителя `|` (в отличие от обычных фраз)

**Пример:**
```
##BLOCK story | Короткий рассказ

ES: Me llamo Victor y tengo cuarenta años. Soy programador y trabajo a distancia. Tengo una familia grande y un hermano menor.
RU: Меня зовут Виктор, мне сорок лет. Я программист и работаю удалённо. У меня большая семья и младший брат.
```

**Многострочный story:**
```
##BLOCK story | История

ES: En el mercado de la ciudad, hay muchas frutas y verduras coloridas.
Los vendedores gritan sus precios. Hoy hace buen tiempo, así que hay mucha gente.
Yo compro manzanas y tomates para la cena de esta noche.

RU: На городском рынке много ярких фруктов и овощей.
Продавцы кричат цены. Сегодня хорошая погода, поэтому много людей.
Я покупаю яблоки и помидоры на ужин.
```

**Ошибки:**
```
❌ ES: текст | RU: текст           (нет разделения на строки)
❌ ES: текст                       (нет RU)
❌ RU: текст                       (нет ES)
❌ story: текст на испанском       (нет ES:)
```

## 3. Правила компоновки

### 3.1 Порядок элементов

**Корректный порядок:**
```
#TOPIC (обязателен, один)
  ↓
##BLOCK 1 (первый блок)
  #WORD / #CATEGORY (опционально)
    фраза
    фраза
  #WORD / #CATEGORY (опционально)
    фраза
  ↓
##BLOCK 2 (второй блок)
  ...
  ↓
##BLOCK 3 (третий блок)
  ...
```

**Неправильные порядки:**
```
❌ ##BLOCK ... (без #TOPIC вверху)
❌ #WORD ... #TOPIC ... (тема не первая)
❌ ##BLOCK фраза (фраза сразу после блока, без #WORD/#CATEGORY)
```

### 3.2 Пустые строки

**Правила:**
- ✓ Пустые строки между элементами пропускаются (автоматически)
- ✓ Рекомендуется добавлять пустые строки для читаемости
- ✗ Пустые строки внутри story теряются (story = один блок текста)

**Пример (рекомендуемое форматирование):**
```
#TOPIC 4 | Рассказ о себе

##BLOCK verb_group | Кто я — происхождение

#WORD llamarse | зваться
Me llamo Victor. | Меня зовут Виктор.
¿Cómo te llamas tú? | Как тебя зовут?

#WORD tener | иметь
Tengo cuarenta años. | Мне сорок лет.

##BLOCK phrase_group | Ходовые фразы

#CATEGORY Первое знакомство
Mucho gusto, soy Victor. | Очень приятно, я Виктор.

##BLOCK vocabulary | Ключевая лексика
el programador | программист
la familia | семья
```

### 3.3 Чувствительность к регистру

- ✓ `#TOPIC` (заглавные буквы)
- ✓ `##BLOCK` (заглавные буквы)
- ✓ `#WORD` (заглавные буквы)
- ✓ `#CATEGORY` (заглавные буквы)
- ✗ `#topic` (не работает)
- ✗ `#Category` (не работает)

**Типы блоков чувствительны:**
- ✓ `verb_group` (нижний регистр)
- ✓ `phrase_group` (нижний регистр)
- ✓ `vocabulary` (нижний регистр)
- ✓ `story` (нижний регистр)
- ✗ `Verb_Group` (не работает)

## 4. Частые ошибки парсера и как их исправить

### Ошибка: "Фраза без разделителя на строке X"

**Сообщение:**
```
⚠ Ошибка на строке 15:
  Фраза без разделителя |
  "Me llamo Victor y Victor"
```

**Причина:** забыли ` | ` между испанским и русским текстом.

**Исправление:**
```
❌ Me llamo Victor y VictorМеня зовут Виктор

✓ Me llamo Victor y Victor. | Меня зовут Виктор.
```

---

### Ошибка: "Неверный разделитель (нужен пробел)"

**Сообщение:**
```
⚠ Ошибка на строке 18:
  Разделитель без пробелов
  "Me llamo Victor|Меня зовут"
```

**Причина:** разделитель `|` без пробелов.

**Исправление:**
```
❌ Me llamo Victor|Меня зовут    (нет пробелов)
❌ Me llamo Victor | Меня зовут  (только слева)
❌ Me llamo Victor| Меня зовут   (только справа)

✓ Me llamo Victor | Меня зовут   (пробелы с обеих сторон)
```

---

### Ошибка: "Блок не найден перед фразой"

**Сообщение:**
```
⚠ Ошибка на строке 12:
  Фраза без предшествующего блока или группы
  "Me llamo Victor. | Меня зовут Виктор."
```

**Причина:** фраза идёт сразу после ##BLOCK, без #WORD или #CATEGORY.

**Исправление:**
```
❌ ##BLOCK verb_group | Глаголы
   Me llamo Victor. | Меня зовут Виктор.

✓ ##BLOCK verb_group | Глаголы
  #WORD llamarse | зваться
  Me llamo Victor. | Меня зовут Виктор.
```

---

### Ошибка: "Двойной разделитель на строке"

**Сообщение:**
```
⚠ Ошибка на строке 20:
  Две фразы на одной строке
  "Me llamo Victor. | Меня зовут | extra"
```

**Причина:** больше одного `|` на строке.

**Исправление:**
```
❌ Me llamo Victor. | Меня зовут | Вик   (два разделителя)

✓ Me llamo Victor. | Меня зовут Вик     (один разделитель)
```

---

### Ошибка: "Рассказ без RU или ES"

**Сообщение:**
```
⚠ Ошибка на строке 45:
  Story блок неполный: ES найден, но RU не найден
```

**Причина:** забыли `RU:` после `ES:`.

**Исправление:**
```
❌ ##BLOCK story | Рассказ
   ES: Me llamo Victor y tengo cuarenta años.

✓ ##BLOCK story | Рассказ
  ES: Me llamo Victor y tengo cuarenta años.
  RU: Меня зовут Виктор, мне сорок лет.
```

---

### Ошибка: "Пустой текст в фразе"

**Сообщение:**
```
⚠ Ошибка на строке 30:
  Текст на испанском пуст
  "| Меня зовут Виктор."
```

**Причина:** забыли испанский текст.

**Исправление:**
```
❌ | Меня зовут Виктор.              (нет испанского)

✓ Me llamo Victor. | Меня зовут Виктор.
```

## 6. Соответствие с lesson.json

Когда вы загружаете текстовый файл в Combine, он парсится в структуру JSON. Вот как элементы разметки соответствуют полям lesson.json:

### 6.1 Иерархия в JSON

```json
{
  "topic_id": "01-primer-saludo",     // из #TOPIC номер + название (автогенерирован)
  "topic_number": 1,                   // из #TOPIC номер
  "title_ru": "Первое знакомство",     // из #TOPIC название
  
  "blocks": [
    {
      "block_id": "b1",                // автогенерирован (b1, b2, ...)
      "type": "verb_group",            // из ##BLOCK тип
      "title_ru": "Главные глаголы",   // из ##BLOCK название
      "order_index": 0,                // порядок блока
      
      "groups": [                      // для verb_group, phrase_group
        {
          "key": "llamarse",           // из #WORD инфинитив
          "translation_ru": "зваться", // из #WORD перевод
          "order_index": 0,
          
          "phrases": [                 // фразы под этим #WORD
            {
              "id": "01-b1-llamarse-01",  // автогенерирован
              "es": "Hola, me llamo Juan.",
              "ru": "Привет, меня зовут Хуан.",
              "audio": { ... },           // добавляется при генерации
              "duration_ms": { ... }      // добавляется при генерации
            },
            { ... }
          ]
        }
      ]
    },
    {
      "block_id": "b2",
      "type": "vocabulary",
      "title_ru": "Слова",
      
      "words": [                       // для vocabulary (без группировки)
        {
          "id": "01-b2-vocab-01",
          "es": "la casa",
          "ru": "дом",
          "audio": { ... },
          "duration_ms": { ... }
        }
      ]
    },
    {
      "block_id": "b3",
      "type": "story",
      "title_ru": "История",
      
      "text_es": "Me llamo Juan...",    // для story (не фраза, сплошной текст)
      "text_ru": "Меня зовут Хуан...",
      "audio": { ... },
      "duration_ms": { ... }
    }
  ]
}
```

### 6.2 Идентификаторы (ID)

Все ID автогенерируются в формате:
```
<тема>-[b<блок>]-<группа>-<номер>

Примеры:
01-b1-llamarse-01   (тема 01, блок 1, группа llamarse, фраза 1)
01-b2-vocab-01      (тема 01, блок 2, vocabulary, слово 1)
01-b3-story         (тема 01, блок 3, story - нет номера)
```

## 7. Кодировка и символы

### Поддерживаемые символы

- ✓ Испанский: буквы, диакритика (á, é, í, ó, ú, ñ, ü)
- ✓ Русский: буквы, ударения (опционально)
- ✓ Пунктуация: . , ! ? : ; " ' - ( ) [ ] ...
- ✓ Цифры: 0–9
- ✓ Апостроф: ' (для испанского d'accord и т.д.)
- ✗ Специальные символы (†, ‡, ℋ, и т.д.)
- ✗ Emoji (😊)

### Кодировка файла

- ✓ UTF-8 (рекомендуется)
- ✓ UTF-16 (поддерживается)
- ✗ Latin-1 (может быть проблема с диакритикой)

**Проверка в macOS:**
```bash
file lesson.txt
# → lesson.txt: UTF-8 Unicode text
```

## 8. Ограничения и лимиты

| Параметр | Лимит | Примечание |
|----------|-------|-----------|
| **Фраз на тему** | 0–500 | Рекомендуется 30–100 |
| **Слов в vocabulary** | 0–200 | Рекомендуется 10–50 |
| **Символов в фразе** | 1–500 | Рекомендуется 10–100 |
| **Символов в рассказе** | 1–5000 | Рекомендуется 100–1000 |
| **Блоков на тему** | 1–20 | Рекомендуется 3–5 |
| **Групп в блоке** | 1–100 | Рекомендуется 3–10 |

## 10. Валидация перед генерацией

Перед тем как нажать "Генерировать" в Combine, проверьте:

- [ ] Файл содержит ровно один `#TOPIC`
- [ ] Каждый `#WORD` идёт только внутри `verb_group`
- [ ] Каждый `#CATEGORY` идёт только внутри `phrase_group`
- [ ] Каждая фраза содержит ровно один ` | `
- [ ] Все `##BLOCK` типы написаны правильно
- [ ] `story` блок содержит обе строки (ES: и RU:)
- [ ] Нет пустых фраз (текст с обеих сторон от `|`)
- [ ] Файл сохранён в UTF-8

**Быстрая проверка в Combine:**
1. На экране "Импорт текста"
2. Загрузите файл
3. Посмотрите счётчик: "Ошибок: 0" ✓
4. Если ошибок > 0 → исправьте по подсказкам
5. Кликните на красную линию → Combine покажет номер строки и тип ошибки
